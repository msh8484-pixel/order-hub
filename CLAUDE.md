## 핵심 기능 (MVP)

### 1. 온라인 주문 수집
- 6개 플랫폼 API 연동 (API 없는 곳은 RPA/Vision)
- 날짜 지정 주문(@표시) 자동 분류 및 필터링
- 당일 처리 주문만 생산 목록에 반영

### 2. 매장 발주 입력 (모바일)
- 각 지점 점장이 마감 시간 전 모바일로 수량 입력
- 매장별 개별 페이지
- Google Sheets 완전 대체

### 3. 생산팀 대시보드
- 당일 생산 목록 시간순 정렬
- 총 수량 자동 합산
- 태블릿에서 처리 완료 체크

### 4. 사장님 대시보드
- 달력 뷰 — 날짜별 주문 현황
- 채널별/매장별 주문 통계
- 매출/출고 데이터 누적

## 2단계 기능 (MVP 이후)
- 수기 주문서 사진 → OCR 자동 입력
- 고객 CRM (재구매, 연락처 누적)
- 택배사 API → 송장번호 자동 문자 발송
- 원가 계산 + 제품별 수익률 분석

## 페이지 구조
- / → 로그인
- /dashboard → 사장님 대시보드 (달력 + 전체 현황)
- /store/[id] → 매장별 발주 입력 페이지
- /production → 생산팀 대시보드
- /orders → 온라인 주문 목록 (날짜 필터)
- /api/orders → 주문 수집 API
