## 4. 분석 인사이트

### 4-1. 상품 구조 특이사항
- **상품명에 갯수가 포함됨** 
  - "떡함지 쑥떡 쑥인절미 20개" → 1주문 = 20개 생산
  - "갯수 : 30개" 옵션 → 실제 30개 생산
  - order-hub에서 실제 생산 수량 계산 시 별도 로직 필요
  
- **동일 품목 갯수별 SKU 분리** (쑥인절미 10/20/30/55개)
  - 생산 시 "쑥인절미" 총 몇 개인지 합산 필요
  - ex) 20개 40건 + 30개 16건 + 55개 2건 = 800+480+110 = 1,390개

### 4-2. 주력 상품군
1. **쑥떡 쑥인절미 시리즈** — 압도적 1위 (갯수 옵션 합산 시 전체의 약 60%)
2. **약밥/약식** — 2위
3. **무설탕 시리즈** — 건강 라인, 성장 중
4. **쑥찹쌀떡** — 꾸준한 수요
5. **답례떡** — 대량 주문 (70박스 한 건)

### 4-3. 고객 특성
- B2B 대량 주문 명확 (신한PWM팀장 등 기업 구매자)
- 개인결제창 1,388개 = B2B 납품이 핵심 채널
- VVIP(120건), VIP(93건) 고빈도 재구매 고객 존재
